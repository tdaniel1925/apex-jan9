License FAQ

If I need to use the template on a second site, do I need a second license?

Yes. One template purchase allows you to use it on one finished site. If you need to use it on a second site, you must purchase a second license.

Can I use the template on a test site and then move it to a live site when finished?

Yes, you can. You can keep the template on a test site to build it, then move it to the live site once you’re finished.

Can I keep a test and live sites with one Regular License purchase?

You can keep one test/staging website and one live website with one Regular License. Keep in mind that you must use the same domain.

Installation

Follow the below steps to get started with Optive:

  1. Unzip the downloaded package and open the /html folder to find all the template files. You will need to upload these files and sub folders to your hosting web server using FTP or file manager in order to use it on your website.
  2. Below is the folder/files structure and needs to be uploaded to your website root directory:
    • css - Stylesheets Folder
    • webfonts - Icon Font Folder
    • images - Demo Images Folder
    • js - Javascripts Folder
    • form-process.php - Contact Form Process File
    • html/*.html - All the templates
  3. Edit the files using a text based editor like VS Code or Sublime text.
  4. Upload your files to your servers in the public_html folder (Apache) or /var/www/html Folder (Nginx).

Color Schemes

You can change your Website’s Color Scheme in an instant. You simply need to set the your Color Code in the css/custom.css file using the following code:

Code Example

								:root{
	--primary-color		: #0A1119;
	--secondary-color	: #FFFFFF;
	--bg-color		: #F5F5F5;
	--text-color		: #4B535D;
	--accent-color		: #097C7D;
	--accent-secondary-color: #20282D;
	--white-color		: #FFFFFF;
	--divider-color		: #0A11191A;
	--dark-divider-color	: #FFFFFF1A;
	--error-color		: rgb(230, 87, 87);
}

Changing Fonts

  • Using Google Fonts

    
<link href="https://fonts.googleapis.com/css2?family=Mona+Sans:ital,wght@0,200..900;1,200..900&family=Public+Sans:ital,wght@0,100..900;1,100..900&display=swap" rel="stylesheet">

    In order to change the Fonts, you will need to Edit the Above Links with your Chosen Google Fonts.

  • Using Self-Hosted Fonts

    
@font-face {
  font-family: 'MyWebFont';
  src: url('webfont.woff2') format('woff2'), /* Super Modern Browsers */
       url('webfont.woff') format('woff'), /* Pretty Modern Browsers */
       url('webfont.ttf')  format('truetype'), /* Safari, Android, iOS */
       url('webfont.svg#svgFontName') format('svg'); /* Legacy iOS */
}

Template Customization

To customize the template, add a new my-style.css file in the CSS folder and include it in the <head> section of your HTML file. This approach helps you manage your customizations more effectively when updating the template to the latest version.

Code Example

								<head>
    <link rel="stylesheet" href="css/my-style.css" type="text/css" />
</head>

Sticky Header

To create a sticky header, add the active-sticky-header class to the <header> tag.

Code Example

								<header class="main-header active-sticky-header">

...

</header>
